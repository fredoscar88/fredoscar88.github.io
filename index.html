Changelog:

  • -Added general properties file (MSP.properties)
  • -You can set your own server starting arguments in the properties file
  • -MSP can now be run without the batch file, but only if you do not want to have access to the MSP Console window
  • -Added "INIT" feature
  • -Redstone is now set into an output file, that can be called upon again to replace the redstone with ReplaceRedstone in the console or !RplRdStone ingame
  • -MSP now only sends one packet to the server instead of several when setting redstone
  • -Server should no longer "hang" with 20hz "/say"s
  • -General ground work to prepare for changes to how the redstone is calculated
  • -Probably some other stuff I'm forgetting
